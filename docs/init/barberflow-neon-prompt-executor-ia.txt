REGRAS DO MVP
- Sistema de agendamentos para barbearia.
- 2 barbeiros.
- Cliente escolhe barbeiro.
- Cliente escolhe serviço.
- Serviço tem preço e duração.
- Agendamento nasce como pendente.
- Admin confirma/cancela/conclui/cria encaixe.
- Cliente cancela por token.
- Cliente remarca por token.
- Feriados e bloqueios impedem agenda.
- Google Calendar só cria evento ao confirmar.
- Sem credenciais Google: usar modo simulado.
- WhatsApp é link manual.
- Sem pagamento online.
- Sem multiempresa.
- Admin único por ADMIN_TOKEN.
