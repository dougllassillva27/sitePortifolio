BARBERFLOW NEON — PASSO A PASSO PARA IDE/IA

OBJETIVO
Recriar o BarberFlow usando Neon Postgres, HTML, CSS, JavaScript e Python/FastAPI.

ARQUITETURA
Frontend:
- HTML
- CSS
- JavaScript puro
- Netlify

Backend:
- Python
- FastAPI
- Psycopg 3
- Render

Banco:
- Neon Postgres

REGRA CRÍTICA
Nunca conectar o frontend direto no Neon.
A DATABASE_URL fica apenas no backend.

COMO CRIAR
1. Execute criar-barberflow-neon-estrutura.bat em uma pasta vazia.
2. Entre na pasta:
   cd barberflow-neon

FRONTEND
1. npm install
2. npm run test
3. npm run build
4. npm run dev

BACKEND
1. cd backend
2. python -m venv .venv
3. .venv\Scripts\activate
4. pip install -r requirements.txt
5. copy .env.example .env
6. Edite backend/.env e informe DATABASE_URL do Neon.
7. python scripts/aplicar_migracoes.py
8. uvicorn app.main:app --reload

VALIDAÇÃO
- Frontend: http://localhost:4173
- Backend: http://localhost:8000/saude
- API docs: http://localhost:8000/docs

PROMPT PARA IDE
Execute como IA executora.
Leia docs/escopo-mvp.md, docs/regras-de-negocio.md, docs/arquitetura.md e docs/checklist-testes.md.
Não apenas oriente.
Implemente a próxima tarefa pendente, rode testes, rode build e corrija falhas.
Use Neon, HTML, CSS, JavaScript e Python.
Não use Supabase.
Não exponha DATABASE_URL no frontend.
